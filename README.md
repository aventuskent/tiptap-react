# tiptap-react

Super light rich text editor based on TipTap and React

![avatar](/public/tiptap.png)

The following features are currently available
1. Bold
2. BulletList
3. Align
4. ColorPane
5. Image
6. Italic
7. OrderedList
8. Redo
9. Retract
10. SizePane
11. Table
12. Underline
13. Undo

Under continuous update......
